Frequently Asked Questions


What is this website about ?

This is a website that helps educational institutions create, conduct and host computer lab sessions for their students. Here, professors can create new lab questions, students can participate in their allocated labs and admins can monitor the labs.

How do we use the website ?

All activities within the website can only be accessed after loging in using the unique ID, once logged in, access to specific roles like student/professor is granted based on the access specifications of the ID used. Thereafter, the user may go about using the features of their respective role given to them by our interactive and easy to use custom dashboards.
